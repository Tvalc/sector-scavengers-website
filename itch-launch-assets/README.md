# itch.io demo launch assets

Everything needed to ship the **Sector Scavengers: Signal & Salvage** demo as a free, public, HTML5 browser-playable page on itch.io. Soft cap on session length: 3 hours.

## Specs (locked)

- Build type: HTML5, browser-playable, no install.
- Pricing: Free, fully public.
- Visibility flow: Draft, then Restricted (press week), then Public on launch day.
- Scope: Open run, soft-capped at 3 hours of session time.
- Timeline: 1 month, quality over speed.
- Source build: `Sector-Scavengers-SS-Redo` (HTML5 single-page build).

## Files in this folder

1. `01_PAGE_DESCRIPTION.md`: paste this directly into the itch.io description editor (Markdown mode).
2. `02_PUBLISH_RUNBOOK.md`: every field, in order, with the exact value to enter.
3. `03_ASSET_PREP_CHECKLIST.md`: which Steam asset maps to which itch.io slot, with crop sizes.
4. `04_LAUNCH_DEVLOG_POST.md`: devlog post to publish the moment the page flips public.
5. `05_OUTREACH_BLURB.md`: short DM for press and streamers during the restricted week.
6. `06_TIMELINE.md`: the 1-month plan, week by week.
7. `07_NOTES_AND_FLAGS.md`: open questions, AI disclosure copy, things to verify before going public.

## Brand rules enforced across this folder

- No em dashes anywhere. Commas, colons, or periods only.
- Never use the words: straightforward, genuinely, honestly.
- Product names: Art Studio, Code Studio, Collections. Makko AI, not Macco / Maco / MCO.
- CTA buttons: background `#3b8edf`, white text `#ffffff`. Inline links: `#3b8edf`.
- Steam wishlist link is the primary outbound CTA on the page.
